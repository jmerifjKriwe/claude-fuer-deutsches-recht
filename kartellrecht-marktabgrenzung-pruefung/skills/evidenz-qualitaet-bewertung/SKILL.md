# Evidenz-Qualitätsbewertung

## Grundprinzip

Eine Marktabgrenzung ist nur so stark wie die sie stützenden Belege. Die EU-Bekanntmachung 2024 unterscheidet zwischen quantitativen und qualitativen Belegen und verlangt eine kohärente, vollständige Darstellung.

## Evidenztypen und ihre Bewertung

### 1. Interne Unternehmensdokumente

**Qualität:** Sehr hoch — spiegeln tatsächliche Geschäftssicht wider.

Relevante Dokumente:
- Marktanalysen für Investoren / Vorstand.
- Wettbewerbsbeobachtungs-Berichte.
- Produktentwicklungs-Strategien (zeigen wahrgenommene Substitute).
- Preissetzungsdokumente (wer wird als Preisreferenz genutzt?).

**Risiko:** Selektive Vorlage von Dokumenten, die die gewünschte Marktabgrenzung stützen.

Prüfung: Gibt es widersprüchliche interne Dokumente, die in der Vorlage fehlen?

### 2. Kundenverhaltensdaten

- Warenkorbanalysen (werden Produkte zusammen oder alternativ gekauft?).
- Kundenbefragungen zu Wechselabsichten (Diversion Surveys).
- Loyalitäts- und Abwanderungsdaten.

**Methodische Anforderungen:** Repräsentativität der Stichprobe, Framing der Befragungsfragen.

### 3. Marktdaten und Marktforschungsberichte

- Branchenberichte (GfK, Nielsen, Gartner, IDC): Wie definieren sie den Markt?
- Abweichung von externen Berichten von der vorgelegten Abgrenzung? → Erklärung erforderlich.

### 4. Elastizitäten und Diversion Ratios

Siehe vertiefend: Skill `elastizitaeten-diversion-ratios`.

Prüffragen:
- Wurden ökonometrische Methoden offengelegt?
- Wurde auf Endogenitätsprobleme eingegangen?
- Datenperiode repräsentativ (kein Sondermarktphase)?

### 5. Triangulationsprinzip

Eine starke Marktabgrenzung stützt sich auf mehrere unabhängige Evidenzquellen, die übereinstimmen. Warnsignal: Nur eine einzige Datenquelle.

## Red-Flag-Liste Evidenz

| Red Flag | Beschreibung |
|----------|-------------|
| Cherry Picking | Nur günstige Studien zitiert |
| Fehlende Quellenangaben | Behauptungen ohne Nachweis |
| Veraltete Daten | Marktdaten > 3 Jahre alt |
| Nicht-repräsentative Stichprobe | Kundenbefragung nur bei loyalen Kunden |
| Widerspruch zu internen Dokumenten | Externe Abgrenzung weicht von internem Marktverständnis ab |
| Fehlende Elastizitäts-Schätzung | SSNIP-Test ohne Datenbasis |

## Bewertungsskala
