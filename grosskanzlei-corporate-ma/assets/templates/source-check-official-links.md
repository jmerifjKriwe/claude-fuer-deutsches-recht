# Official Source Check für Corporate/M&A

| Thema | Primaerquelle für Live-Check | Was prüfen? |
| --- | --- | --- |
| Umwandlung | Gesetze im Internet / UmwG | Form, Verfahren, Register, Fristen, Gläubigerschutz |
| Umwandlungssteuer | Gesetze im Internet / UmwStG | Buchwertfortfuehrung, Sperrfristen, Verlustvortraege |
| StaRUG / Insolvenzplan | Gesetze im Internet / StaRUG und InsO | Planfaehigkeit, Gruppen, Vergleichsrechnung, Restrukturierungsinstrumente |
| Fusionskontrolle | Bundeskartellamt und EU-Kommission | Schwellen, Vollzugsverbot, Filing-Timing |
| FDI / Investitionspruefung | BMWK / AWV | Sektor, Erwerbsschwellen, Meldepflicht, Untersagungsrisiko |
| FSR | EUR-Lex / EU-Kommission | Drittstaatliche Finanzbeitraege, Notifizierung, Timing |
| Public M&A / MAR | BaFin / EUR-Lex | Angebotsunterlagen, Ad-hoc, Insiderlisten, Aufschub |
