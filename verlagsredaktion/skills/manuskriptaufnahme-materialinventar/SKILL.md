# Manuskriptaufnahme und Materialinventar

## Ziel

Vor jeder Bearbeitung muss klar sein, welches Material existiert, woher es kommt und wofür es verwendet werden darf.

## Inventar

| Feld | Inhalt |
| --- | --- |
| Nr. | Laufende Nummer |
| Typ | Manuskript, Mail, Folie, Bild, Tabelle, Entscheidung, Vertrag |
| Herkunft | Autor:in, Verlag, Dritte, amtliche Quelle |
| Nutzbarkeit | frei, unklar, gesperrt |
| Aufgabe | lesen, zitieren, redigieren, Rechte klären |

## Output

- Materialinventar.
- Lückenliste.
- Rechteampel.
- Vorschlag für Bearbeitungsreihenfolge.
