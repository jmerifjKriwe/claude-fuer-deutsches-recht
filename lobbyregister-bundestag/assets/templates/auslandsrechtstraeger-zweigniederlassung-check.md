# Auslandsrechtstraeger und unselbststaendige Zweigniederlassung

| Prüfpunkt | Befund | Beleg | Risiko | Naechste Handlung |
|---|---|---|---|---|
| Ausländischer Rechtstraeger existiert |  |  |  |  |
| Deutsche Zweigniederlassung ist unselbststaendig |  |  |  |  |
| Handelsregistereintragung DE |  |  |  |  |
| Bank-/Aufsichtserlaubnis oder Passporting |  |  |  |  |
| Wer betreibt Interessenvertretung tatsaechlich |  |  |  |  |
| Kontaktpersonen in Deutschland |  |  |  |  |
| Hauptstadtrepraesentanz/Buero |  |  |  |  |
| Ein Registereintrag des Rechtstraegers deckt die Zweigniederlassung ab |  |  |  |  |
| Separater Zweigniederlassungs-Eintrag waere irrefuehrend |  |  |  |  |

**Arbeitsregel:** Die Zweigniederlassung nicht vorschnell als eigene juristische Person behandeln. Wenn unklar bleibt, ob das Portal eine gesonderte technische Darstellung erwartet, Entwurf einer Anfrage an die registerfuehrende Stelle erstellen.
