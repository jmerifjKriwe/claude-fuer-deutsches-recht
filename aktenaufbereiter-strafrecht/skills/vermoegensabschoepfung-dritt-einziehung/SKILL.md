# Vermoegensabschoepfung Dritt Einziehung

## Arbeitsbereich

**Vermoegensabschoepfung Dritt Einziehung** ordnet den Fall über die tragenden Prüffelder: Spezialfall Dritt-Arrest in der Vermoegensabschoepfung §§, Vermoegensabschoepfung §§ 73 ff, Verstaendigung im Strafverfahren § 257c StPO vorbereiten. Zuerst wird das Feld bestimmt, das die Akte wirklich trägt; ergänzende Felder kommen nur hinzu, wenn sie dieselbe Frist, Zuständigkeit, Beweislast oder denselben Output berühren.
