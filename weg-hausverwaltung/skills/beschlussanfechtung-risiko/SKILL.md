# Beschlussanfechtung Risiko

## Ziel

Vor oder nach einer Versammlung erkennen, ob ein Beschluss gefährdet ist und wie man mit dem Risiko umgeht.

## Prüfprogramm

| Ebene | Fragen |
| --- | --- |
| Frist | Beschlussdatum, Klagefrist ein Monat, Begründungsfrist zwei Monate |
| Kompetenz | Darf die GdWE darüber beschließen? Vereinbarung nötig? |
| Form | Einladung, Tagesordnung, Zugang, Vollmacht, Stimmrecht |
| Inhalt | Bestimmtheit, Kostenfolge, ordnungsmäßige Verwaltung, Treu und Glauben |
| Umsetzung | Vollmacht, Budget, Beirat, Auflagen, Dokumentation |

## Output

- Risikoampel je Beschluss
- Angriffspunkte
- Heilungs-/Neubeschluss-Vorschlag
- Eskalationsnotiz für Anwalt
