# Mandat- und Objekt-Triage

## Ziel

Aus einer ungeordneten Verwaltungsakte entsteht ein arbeitsfähiges Objektprofil mit Fristen, Zuständigkeiten, Dokumentenlage und nächstem Skill.

## Abfrage

| Bereich | Klären |
| --- | --- |
| Objekt | Adresse, Einheiten, Gewerbeanteil, Baujahr, Verwaltung seit wann |
| Rollen | GdWE, Verwalter, Beirat, einzelne Eigentümer, vermietende Eigentümer, Mieter |
| Grundakte | Teilungserklärung, Gemeinschaftsordnung, Aufteilungsplan, Beschlusssammlung |
| Verwaltung | Verwaltervertrag, Vollmachten, Beiratsbestellung, Sonderzuständigkeiten |
| Finanzen | Wirtschaftsplan, Jahresabrechnung, Rücklagen, Hausgeldrückstände, Sonderumlagen |
| Bau/Technik | Instandhaltungsstau, Angebote, Gutachten, Gewährleistungsfristen |
| Streit | laufende Beschlussklagen, Beschwerden, Störungen, Datenschutzprobleme |

## Arbeitsweise

1. Dokumente benennen und Lücken markieren.
2. Fristen und irreversible Risiken nach oben ziehen.
3. Vorgänge in Körbe sortieren: Versammlung, Beschluss, Abrechnung, Bau, Hausgeld, Kommunikation, Gericht.
4. Primären Folge-Skill vorschlagen.

## Output

- `objektprofil.md`
- `fristen-und-risiken.md`
- `dokumentenluecken.md`
- Skill-Routing mit Priorität
