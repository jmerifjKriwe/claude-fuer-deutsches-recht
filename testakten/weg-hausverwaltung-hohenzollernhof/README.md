# Testakte: WEG Hohenzollernhof — Hausverwaltung unter Druck

Fiktive Testakte für das Plugin `weg-hausverwaltung`.

Die Akte simuliert eine typische, leicht überforderte Hausverwaltung einer Berliner WEG mit 38 Wohnungen, zwei Gewerbeeinheiten, angespanntem Beirat, einer bevorstehenden Eigentümerversammlung, undichten Balkonen, drei Handwerkerangeboten, Streit um Steckersolargeräte, einer Wallbox, Hausgeldrückständen, einer fehleranfälligen Jahresabrechnung und vielen Eigentümermails.

## Dateien

| Datei | Inhalt |
| --- | --- |
| `01-verwaltungsbeirat-notiz.md` | Beiratsnotiz mit Prioritäten, Fristen und Stimmungslage |
| `02-einladung-eigentuemerversammlung-entwurf.md` | Rohentwurf einer Einladung mit problematischen Tagesordnungspunkten |
| `03-handwerkerangebote-balkone-und-dach.md` | Drei schwer vergleichbare Angebote für Balkone/Dachrinne |
| `04-jahresabrechnung-2025-rohpruefung.md` | Jahresabrechnung mit Rückfragen, Kostenpositionen und Verteilerschlüsseln |
| `05-eigentuemer-und-mieterbeschwerden.md` | Fragmentarische Beschwerdemails, Hausordnungsstreit, Mieter als Störer |
| `06-beschlussliste-alt-und-neu.md` | Alte Beschlüsse, offene Umsetzung, neue Beschlussideen |

## Ziel

Die Akte soll ohne Musterlösung genutzt werden. Gute Tests:

- Eigentümerversammlung vorbereiten.
- Beschlussvorlagen für Balkonsanierung, Sonderumlage, Steckersolar und Wallbox schreiben.
- Anfechtungsrisiken erkennen.
- Jahresabrechnung und Betriebskosten-Schnittstelle kontrollieren.
- Handwerkerangebote vergleichbar machen.
- Beirat und Verwalter in eine saubere Eskalationsstruktur bringen.
