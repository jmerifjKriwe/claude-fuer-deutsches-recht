# Plugin-Gruppen

Thematische Einstiegsknoten für Rechtsgebiete, in denen mehrere Einzelplugins zusammenarbeiten. Die installierbaren Plugins bleiben in ihren eigenen Verzeichnissen; die Gruppen ordnen nur den Einstieg.

| Gruppe | Wofür |
| --- | --- |
| [`insolvenzrecht`](./insolvenzrecht/) | Insolvenz, Sanierung, StaRUG, Fortbestehensprognose, Liquiditätsplanung, Forderungsanmeldung, Insolvenzverwaltung, Verbraucherinsolvenz und Cross-Border-Bankruptcy. |
